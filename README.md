A desktop-based Student Course Management System developed using Java Swing for the graphical user interface and MySQL for the database. This project allows users to manage students, courses, enrollments, and grades through an intuitive GUI. It demonstrates key DBMS concepts including primary keys, foreign keys, 1:N & M:N relationships, normalization, views, and stored procedures.

Use Cases:
Educational institutions for managing student-course data
Demonstrates application of DBMS concepts in a real-world scenario
